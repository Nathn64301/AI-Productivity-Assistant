# AI-Productivity-Assistant.
An AI-powered productivity assistant that helps users manage tasks, schedules, notes, and workflows efficiently.
## Problem Statement
This AI tool is built for summarizing meeting notes for online and onsite meetings.
## Solution Overview 
It makes the process faster, more accurate, and less stressful. This way, it saves time, reduces human error, and allows the secretary to focus more on participating in the meeting rather than trying to write every detail.
## Tools Used
Loveable
TypeScript 
JavaScript
## Prompt Engineering
My prompt was that I wanted the AI Tool to summarize meeting notes. The summary should be listed with sync of the opening agenda, key points, decisions, action items, and deadlines.
## Challenges & Responsible AI
Made sure that the output/summary is in a formal languge and accurate to the notes
